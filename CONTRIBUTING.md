# Contributing

Contributions should make this knowledge base easier to learn from and safer to
use.

## Add or improve study material

1. Choose the correct module.
2. Explain the topic in plain language before adding advanced detail.
3. Add a small example using a lab placeholder.
4. Add or update a quiz question and flashcard.
5. Add a safe practice task when the topic benefits from hands-on work.
6. Check links and spelling.

## Content rules

- Use Markdown.
- Keep one main idea per paragraph.
- Define acronyms.
- Use authorized lab environments only.
- Do not commit credentials, tokens, personal data, packet captures, or
  proprietary course material.
- Paraphrase concepts in your own words and link to public primary sources.
- Make uncertainty visible instead of guessing.

## Commit suggestions

Use a short, descriptive message, for example:

- docs(module-03): clarify passive and active OSINT
- quiz(module-09): add Kerberos reasoning questions
- template(report): improve remediation guidance

## Review checklist

- [ ] No existing file was unintentionally removed.
- [ ] The change is understandable to a beginner.
- [ ] Commands and targets are clearly marked as lab-only.
- [ ] Answers appear after quiz questions.
- [ ] Links work from the file that contains them.
- [ ] Sensitive data is absent or redacted.
