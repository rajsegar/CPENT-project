# Lab Workspace Guide

Use this directory for lab plans and sanitized learning summaries. Start from
[the shared lab template](../Templates/Lab-Template.md).

## Suggested naming

Use a date, module, and short topic:

    2026-06-26-module-03-passive-osint.md

Store screenshots in a matching folder only when they contain no credentials,
tokens, personal data, or third-party information.

## Evidence habits

- Record the date, time, and time zone.
- Record tool versions.
- Save the exact scope and target.
- Capture the minimum evidence needed to prove a result.
- Hash important evidence when integrity matters.
- Redact secrets before committing.
- Record cleanup and retest results.

Raw packet captures and browser archives are ignored by Git because they often
contain sensitive data.
