# Module 01 — Practical Lab

## Lab goal

Understand scope, authorization, and methodology before running tools.

## Use a safe target

Choose one:

- TryHackMe room
- Hack The Box machine
- DefHawk lab
- PortSwigger Web Security Academy lab
- Your own local VM

Do not use random public websites.

## Engagement template

Fill this in before testing:

```text
Target:
Permission:
Scope:
Goal:
Testing window:
Tools allowed:
Tools not allowed:
DoS allowed? No
Social engineering allowed? No, unless explicitly stated
What I will not test:
Emergency stop contact:
```

## Example

```text
Target: TryHackMe room IP
Permission: TryHackMe lab environment
Scope: Only the assigned room IP
Goal: Practice reconnaissance and enumeration
Testing window: Today, 2 hours
Tools allowed: Nmap, browser, curl
Tools not allowed: DoS tools
DoS allowed? No
Social engineering allowed? No
What I will not test: Any public website or unrelated IP
Emergency stop contact: Not applicable in lab
```

## Task

Write a short paragraph:

What phase am I in right now?

Answer should be:

Pre-Attack phase, because I am defining scope, understanding permission, and planning before scanning.
