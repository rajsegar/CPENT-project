# Module 01 — Quiz

## Multiple choice

### 1. What is the main purpose of penetration testing?

A. To create a list of vulnerabilities  
B. To simulate an authorized attack and prove real risk  
C. To replace security monitoring  
D. To damage production systems  

Answer: B

### 2. What separates a penetration tester from a criminal attacker?

A. Tool choice  
B. Programming skill  
C. Written authorization and scope  
D. Speed  

Answer: C

### 3. Which phase includes reconnaissance and scanning?

A. Pre-Attack  
B. Attack  
C. Post-Attack  
D. Reporting only  

Answer: A

### 4. Which phase includes exploitation and privilege escalation?

A. Pre-Attack  
B. Attack  
C. Post-Attack  
D. Contract phase  

Answer: B

### 5. Which phase includes cleanup and final reporting?

A. Pre-Attack  
B. Attack  
C. Post-Attack  
D. Initial access  

Answer: C

### 6. What does PCI DSS mainly relate to?

A. Healthcare data  
B. Payment card data  
C. Cloud federal systems  
D. Financial reporting  

Answer: B

### 7. What does GDPR mainly protect?

A. EU personal data  
B. US federal cloud systems  
C. Wireless networks  
D. Source code  

Answer: A

### 8. What is the main weakness of relying only on vulnerability scanning?

A. It is always illegal  
B. It does not prove real-world attack impact  
C. It cannot detect open ports  
D. It cannot create reports  

Answer: B

### 9. What is lateral movement?

A. Moving from one compromised system to another  
B. Running a vulnerability scanner  
C. Writing the final report  
D. Asking for permission  

Answer: A

### 10. What should be done before any penetration test?

A. Exploit the target  
B. Create backdoor access  
C. Get written authorization and define scope  
D. Clear logs  

Answer: C
